# 용어정리
```
    Epoch: 전체 데이터가 모두 학습한 단위
    Loss(Error): 정답 값과의 오차
    Accuracy: 정확도

    지도학습( Supervised Learning)  <= 입력 출격 모두 존재
        회귀( Regression )
            사람과 사물 분류
            스팸메일 분류
            지문, 홍채인식 판별
        분류( Classification)
            부동산 가격 예측
            주식 가격 예측
            출산 인구 예측

    비지도학습( Unsupervised Learing) <= 입력만 존재
        군집( clustering )

    input Data( x, feature )   ===>  Output Data ( y(label), target ) 

    
```
